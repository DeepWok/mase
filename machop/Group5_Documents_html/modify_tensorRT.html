modify_tensorRT.create_new_module_tensorRT(mase_op: str, original_module: Module, config: dict, node_meta: dict, baseline_module: Module | None = None, successor_module: Module | None = None, input_layers=None, output_layers=None)

Create a new quantized module for TensorRT based on the original module and the operation type. Only Supports Fake Quantization.

Parameters:

  • mase_op (str) – The operation type, either “linear” or “conv2d”.

  • original_module (nn.Module) – The original module to be modified.

  • config (dict) – The configuration dictionary containing the quantization parameters.

  • QUANTIZE_DESC_INPUT_LAYER – The quantization descriptor for the input tensor.

Raises:

NotImplementedError – If the operation type is not supported.

Returns:

The new module with the specified operation type and quantization parameters.

Return type:

nn.Module
